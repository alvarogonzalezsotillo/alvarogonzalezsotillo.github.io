Blogueando con orgmode

Emacs no es solo un editor de texto. Una vez que te decides a probarlo, no puedes parar de pensar en qué más puedo hacer sin salir de Emacs. Este sitio web está hecho con Orgmode, un major mode de Emacs que permite generar documentos a partir de ficheros de texto plano (como markdown, aunque con más funcionalidad).

Basados en Orgmode, existen varios sistemas para generar un sitio web, y después de probar algunos me he decidido por org-page, aunque con un tema propio.
